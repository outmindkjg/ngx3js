The goal of this section is to give a brief introduction to ngx3js. We will start by setting up a scene, with a spinning cube. A working example is provided at the bottom of the page in case you get stuck and need help.

Before we start

Before you can use ngx3js, you need somewhere to display it. Save the following angular html to a file as angular template on your computer, along with a install npm of [link:https://www.npmjs.com/package/ngx3js ngx3js] in the working directory, and open it with command ng serve.

src/app/app.module.ts

import { Ngx3JsModule } from 'ngx3js'; @NgModule({ declarations: [ ], imports: [ Ngx3JsModule ], bootstrap: [AppComponent], }) export class AppModule {}

src/app/app.component.html

<div style="width: 700px; height: 500px; display: block; position: relative"> </div>

That's all. All the code below goes into the empty <div> tag.

Creating the scene

To actually be able to display anything with ngx3js, we need three things: scene, camera and renderer, so that we can render the scene with camera.

src/app/app.component.html

<div style="width: 700px; height: 500px; display: block; position: relative"> <ngx3js-renderer [controlType]="'orbit'" > <ngx3js-camera [type]="'PerspectiveCamera'" > <ngx3js-position [x]="-50" [y]="0" [z]="-50"> </ngx3js-position> </ngx3js-camera> <ngx3js-scene #scene> </ngx3js-scene> </ngx3js-renderer> </div>

Let's take a moment to explain what's going on here. We have now set up the scene, our camera and the renderer.

There are a few different cameras in ngx3js. For now, let's use a PerspectiveCamera.

"That's all good, but where's that cube you promised?" Let's add it now.

src/app/app.component.html

<div style="width: 700px; height: 500px; display: block; position: relative"> <ngx3js-renderer [controlType]="'orbit'" > <ngx3js-camera> <ngx3js-position [x]="-50" [y]="0" [z]="-50"> </ngx3js-position> </ngx3js-camera> <ngx3js-scene #scene> <ngx3js-light [type]="'AmbientLight'"> </ngx3js-light> <ngx3js-light [type]="'directional'"> <ngx3js-position [x]="-3" [y]="10" [z]="-10"> </ngx3js-position> </ngx3js-light> <ngx3js-mesh> <ngx3js-geometry [type]="'box'" [width]="30" [height]="30" [depth]="30"> </ngx3js-geometry> <ngx3js-material [type]="'meshlambert'" [color]="'0xff0000'"> </ngx3js-material> </ngx3js-mesh> </ngx3js-scene> </ngx3js-renderer> </div>
