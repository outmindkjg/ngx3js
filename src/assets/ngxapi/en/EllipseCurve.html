EllipseCurve

Creates a 2d curve in the shape of an ellipse. Setting the [page:Number xRadius] equal to the [page:Number yRadius] will result in a circle.

Code Example



const curve = new THREE.EllipseCurve(
0, 0, // ax, aY
10, 10, // xRadius, yRadius
0, 2 * Math.PI, // aStartAngle, aEndAngle
false, // aClockwise
0 // aRotation
);
const points = curve.getPoints( 50 );
const geometry = new THREE.BufferGeometry().setFromPoints( points );
const material = new THREE.LineBasicMaterial( { color : 0xff0000 } );
// Create the final object to add to the scene
const ellipse = new THREE.Line( geometry, material );

Parameters

[parameter:number aX] - – The X center of the ellipse. Default is 0.

[parameter:number xRadius] - – The radius of the ellipse in the x direction. Default is 1.
[parameter:number yRadius] - – The radius of the ellipse in the y direction. Default is 1.
[parameter:number aStartAngle] - – The start angle of the curve in radians starting from the positive X axis. Default is 0.
[parameter:number aEndAngle] - – The end angle of the curve in radians starting from the positive X axis. Default is *2 x Math.PI*.
[parameter:boolean aClockwise] - – Whether the ellipse is drawn clockwise. Default is false.
[parameter:number aRotation] - – The rotation angle of the ellipse in radians, counterclockwise from the positive X axis Default is 0.

This constructor creates a new Curve.

[property:number arcLengthDivisions]Inherited

This value determines the amount of divisions when calculating the cumulative segment lengths of a curve via .getLengths.
To ensure precision when using methods like .getSpacedPoints, it is recommended to increase .arcLengthDivisions if the curve is very large. Default is 200.
