MaterialLoader

A loader for loading a [page:Material Material] in JSON format.
This uses the [page:FileLoader FileLoader] internally for loading files.

Code Example



// instantiate a loader
const loader = new THREE.MaterialLoader();
// load a resource
loader.load(
'path/to/material.json', // resource URL
function ( material ) { // onLoad callback
object.material = material;
},
function ( xhr ) { // onProgress callback
console.log( (xhr.loaded / xhr.total * 100) + '% loaded' );
},
function ( err ) { // onError callback
console.log( 'An error happened' );
}
);

[parameter:any onError] - (optional) Will be called when load errors.
