CameraComponent

CameraComponent

Abstract base class for cameras. This class should always be inherited when you build a new camera.

[property:boolean active]   = true Input Public

The camera is active.

[property:number aspect]   = 1 Input Public

Camera frustum aspect ratio, usually the canvas width / canvas height. Default is *1* (square canvas).

[property:boolean autoClear]Input Public

Defines whether the renderer should automatically clear its output before rendering a frame.

[property:boolean autoRotate]Input Public

The autoRotate of camera component

[property:number bottom]Input Public

Camera frustum bottom plane.

[property:boolean clear]Input Public

Tells the renderer to clear its color, depth or stencil drawing buffer(s).
This method initializes the color buffer to the current clear color value.
Arguments default to *true*.

[property:number clearAlpha]Input Public

The clearAlpha of camera component

[property:ThreeColor clearColor]Input Public

Clear the color buffer. Equivalent to calling [page:WebGLRenderer.clear .clear]( true, false, false ).

[property:boolean clearDepth]Input Public

Clear the depth buffer. Equivalent to calling [page:WebGLRenderer.clear .clear]( false, true, false ).

[property:string controlType]   = 'none' Input Public

The controlType of camera component

Notice - case insensitive.

[property:string|number far]Input Public

Camera frustum far plane. Default is *2000*.
Must be greater than the current value of [page:.near near] plane.

[property:number focalLength]Input Public

Sets the FOV by focal length in respect to the current [page:PerspectiveCamera.filmGauge .filmGauge].
By default, the focal length is specified for a 35mm (full frame) camera.

[property:string|number fov]   = 50 Input Public

Camera frustum vertical field of view, from bottom to top of view, in degrees. Default is *50*.

[property:string|number height]   = '100%' Input Public

The size of height
- type number
fixed size
- type string with include %
relative size from renderer size
for example
in case renderer = 1024
'100%' = 1024, '50%' = 512, '50%-10' = 502, '50%+30' = 542
- type string const refer
width,height,x,y,

[property:number left]Input Public

Camera frustum left plane.

[property:string|number near]Input Public

Camera frustum near plane. Default is *0.1*.
The valid range is between 0 and the current value of the [page:.far far] plane.
Note that, unlike for the [page:PerspectiveCamera], *0* is a valid value for an
OrthographicCamera's near plane.

[property:number orthoSize]   = 400 Input Public

The orthoSize of camera component

[property:AbstractObject3dComponent|Object3D referObject3d]Input Public

The referObject3d of camera component

[property:number right]Input Public

Camera frustum right plane.

[property:any scene]Input Public

The given scene

[property:any[] scenes]Input Public

The given scene

[property:boolean scissorTest]   = false Input Public

Indicates whether the scissor test is active or not.
