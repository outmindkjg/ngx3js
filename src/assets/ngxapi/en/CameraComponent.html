[name]

CameraComponent

Abstract base class for cameras. This class should always be inherited when you build a new camera.

Component

<ngx3js-camera></ngx3js-camera>

Properties - Input

See the base [page:AbstractTweenComponent AbstractTweenComponent] class for common input properties.

See the base [page:AbstractObject3dComponent AbstractObject3dComponent] class for common input properties.

[property:boolean active]

The camera is active.

[property:number aspect]

Camera frustum aspect ratio, usually the canvas width / canvas height. Default is *1* (square canvas).

[property:boolean autoClear]

Defines whether the renderer should automatically clear its output before rendering a frame.

[property:boolean autoRotate]

Input of camera component

[property:number bottom]

Camera frustum bottom plane.

[property:boolean clear]

Tells the renderer to clear its color, depth or stencil drawing buffer(s). This method initializes the color buffer to the current clear color value.
Arguments default to *true*.

[property:number clearAlpha]

Input of camera component

[property:ThreeColor clearColor]

Clear the color buffer. Equivalent to calling [page:WebGLRenderer.clear .clear]( true, false, false ).

[property:boolean clearDepth]

Clear the depth buffer. Equivalent to calling [page:WebGLRenderer.clear .clear]( false, true, false ).

[property:string controlType]

Input of camera component

Notice - case insensitive.

[property:string|number far]

Camera frustum far plane. Default is *2000*.

Must be greater than the current value of [page:.near near] plane.

[property:number focalLength]

Sets the FOV by focal length in respect to the current [page:PerspectiveCamera.filmGauge .filmGauge].

By default, the focal length is specified for a 35mm (full frame) camera.

[property:string|number fov]

Camera frustum vertical field of view, from bottom to top of view, in degrees. Default is *50*.

[property:string|number height]

The viewport of this render target. Vector4.height

[property:number left]

Camera frustum left plane.

[property:any material]

Input of camera component

[property:string|number near]

Camera frustum near plane. Default is *0.1*.

The valid range is between 0 and the current value of the [page:.far far] plane. Note that, unlike for the [page:PerspectiveCamera], *0* is a valid value for an OrthographicCamera's near plane.

[property:number orthoSize]

orthoSize of camera component

[property:AbstractObject3dComponent|Object3D referObject3d]

Input of camera component

[property:number right]

Camera frustum right plane.

[property:any scene]

The given scene

[property:any[] scenes]

The given scene

[property:string|number scissorHeight]

A rectangular area inside the render target's viewport. Fragments that are outside the area will be discarded. Vector4.height

[property:boolean scissorTest]

Indicates whether the scissor test is active or not.

[property:string|number scissorWidth]

A rectangular area inside the render target's viewport. Fragments that are outside the area will be discarded. Vector4.width

[property:string|number scissorX]

A rectangular area inside the render target's viewport. Fragments that are outside the area will be discarded. Vector4.x

[property:string|number scissorY]

A rectangular area inside the render target's viewport. Fragments that are outside the area will be discarded. Vector4.y

[property:string storageName]

Input of camera component

[property:number top]

Camera frustum top plane.

[property:string type]

The type of camera

[property:boolean viewport]

The viewport of this render target.

[property:string viewportType]

The viewport of this render target.

Notice - case insensitive.

[property:string|number width]

The viewport of this render target. Vector4.width

[property:string|number x]

The viewport of this render target. Vector4.x

[property:string|number y]

The viewport of this render target. Vector4.y

[property:string|number zoom]

Gets or sets the zoom factor of the camera. Default is *1*.
