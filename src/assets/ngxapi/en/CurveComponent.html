Properties - Input

See the base [page:AbstractSubscribeComponent AbstractSubscribeComponent] class for common input properties.

[property:boolean aClockwise]

Whether the ellipse is drawn clockwise. Default is *false*.

[property:number aEndAngle]

The end angle of the curve in radians starting from the positive X axis. Default is *2 x Math.PI*.

[property:number aRadius]

The aRadius of curve component

[property:number aRotation]

The rotation angle of the ellipse in radians, counterclockwise from the positive X axis (optional). Default is *0*.

[property:number aStartAngle]

The start angle of the curve in radians starting from the positive X axis. Default is *0*.

[property:number aX]

The X center of the ellipse. Default is *0*.

[property:number aY]

The Y center of the ellipse. Default is *0*.

[property:boolean closed]

Whether the curve is closed. Default is *false*.

[property:string curveType]

Possible values are *centripetal*, *chordal* and *catmullrom*.

Notice - case insensitive.

[property:ThreeVector points][]

array of [page:Vector2 Vector2s].
Creates a Path from the points. The first point defines the offset, then successive points are added to the [page:CurvePath.curves curves] array as [page:LineCurve LineCurves].
If no points are specified, an empty path is created and the [page:.currentPoint] is set to the origin.

[property:number tension]

When [page:.curveType] is *catmullrom*, defines catmullrom's tension.

[property:string type]

The curve type

Notice - case insensitive.
