LinearInterpolant

Linear interpolant
See the [page:LinearInterpolant ngx3js docs] page for details.

Code Example



const interpolant = new THREE.LinearInterpolant(
new Float32Array( 2 ), new Float32Array( 2 ),, n, new Float32Array( 1 ) );
interpolant.evaluate( 0.5 );
