[property:number backgroundAlpha]Input Public

The background-color CSS property sets the background color alpha of an element.

[property:string backgroundClip]Input Public

The background-clip CSS property sets whether an element's background extends underneath its border box, padding box, or content box.

[property:string|number backgroundColor]Input Public

The background-color CSS property sets the background color of an element.

[property:string backgroundImage]Input Public

The background-image CSS property sets one or more background images on an element.

[property:string backgroundPosition]Input Public

The background-position CSS property sets the initial position for each background image. The position is relative to the position layer set by background-origin.

[property:string backgroundPositionX]Input Public

The background-position CSS property sets the initial position for each background image. The position is relative to the position layer set by background-origin.

[property:string backgroundPositionY]Input Public

The background-position CSS property sets the initial position for each background image. The position is relative to the position layer set by background-origin.

[property:string backgroundRepeat]Input Public

The background-repeat CSS property sets how background images are repeated. A background image can be repeated along the horizontal and vertical axes, or not repeated at all.

[property:string backgroundRepeatX]Input Public

The background-repeat CSS property sets how background images are repeated. A background image can be repeated along the horizontal and vertical axes, or not repeated at all.

[property:string backgroundRepeatY]Input Public

The background-repeat CSS property sets how background images are repeated. A background image can be repeated along the horizontal and vertical axes, or not repeated at all.

[property:string backgroundSize]Input Public

The background-size CSS property sets the size of the element's background image. The image can be left to its natural size, stretched, or constrained to fit the available space.

[property:string backgroundSizeX]Input Public

The background-size CSS property sets the size of the element's background image. The image can be left to its natural size, stretched, or constrained to fit the available space.

[property:string backgroundSizeY]Input Public

The background-size CSS property sets the size of the element's background image. The image can be left to its natural size, stretched, or constrained to fit the available space.

[property:string|number border]Input Public

The border shorthand CSS property sets an element's border. It sets the values of border-width, border-style, and border-color.

[property:string|number borderBottom]Input Public

The border-bottom shorthand CSS property sets an element's bottom border. It sets the values of border-bottom-width, border-bottom-style and border-bottom-color.

[property:string|number borderColor]Input Public

The border-color shorthand CSS property sets the color of an element's border.

[property:string borderImage]Input Public

The border-image CSS property draws an image around a given element. It replaces the element's regular border.

[property:string|number borderImageOutset]Input Public

The border-image-outset CSS property sets the distance by which an element's border image is set out from its border box.
The parts of the border image that are rendered outside the element's border box with border-image-outset do not trigger overflow scrollbars and don't capture mouse events.

[property:string borderImageRepeat]Input Public

The border-image-repeat CSS property defines how the edge regions of a source image are adjusted to fit the dimensions of an element's border image.

[property:string|number borderImageSlice]Input Public

The border-image-slice CSS property divides the image specified by border-image-source into regions. These regions form the components of an element's border image.

[property:string borderImageSource]Input Public

The border-image-source CSS property sets the source image used to create an element's border image.

[property:string|number borderImageWidth]Input Public

The border-image-width CSS property sets the width of an element's border image.

[property:string|number borderLeft]Input Public

The border-left shorthand CSS property sets all the properties of an element's left border.

[property:string|number borderRadius]Input Public

The border-radius CSS property rounds the corners of an element's outer border edge. You can set a single radius to make circular corners, or two radii to make elliptical corners.

[property:string|number borderRight]Input Public

The border-right shorthand CSS property sets all the properties of an element's right border.

[property:string borderStyle]Input Public

The border-style shorthand CSS property sets the line style for all four sides of an element's border.

[property:string|number borderTop]Input Public

The border-top shorthand CSS property sets all the properties of an element's top border.

[property:string|number borderWidth]Input Public

The border-width shorthand CSS property sets the width of an element's border.

[property:string|number color]Input Public

The color CSS property sets the foreground color value of an element's text and text decorations, and sets the <currentcolor> value. currentcolor may be used as an indirect value on other properties and is the default for other color properties, such as border-color.

[property:string fontStyle]Input Public

[property:string fontWeight]Input Public

The font-weight CSS property sets the weight (or boldness) of the font. The weights available depend on the font-family that is currently set.

[property:string letterSpacing]Input Public

The letter-spacing CSS property sets the horizontal spacing behavior between text characters. This value is added to the natural spacing between characters while rendering the text. Positive values of letter-spacing causes characters to spread farther apart, while negative values of letter-spacing bring characters closer together.

[property:string|number margin]Input Public

[property:string|number marginLeft]Input Public

[property:number opacity]Input Public

The opacity CSS property sets the opacity of an element. Opacity is the degree to which content behind an element is hidden, and is the opposite of transparency.

[property:string|number padding]Input Public

The text-align CSS property sets the horizontal alignment of the content inside a block element or table-cell box. This means it works like vertical-align but in the horizontal direction.

The text-decoration shorthand CSS property sets the appearance of decorative lines on text. It is a shorthand for text-decoration-line, text-decoration-color, text-decoration-style, and the newer text-decoration-thickness property.

[property:string textIndent]Input Public

[property:string textJustify]Input Public

The text-justify CSS property sets what type of justification should be applied to text when text-align: justify; is set on an element.

The text-size-adjust CSS property controls the text inflation algorithm used on some smartphones and tablets. Other browsers will ignore this property.

[property:string textTransform]Input Public

The text-transform CSS property specifies how to capitalize an element's text. It can be used to make text appear in all-uppercase or all-lowercase, or with each word capitalized. It also can help improve legibility for ruby.
