Sprite

Constructor

[constructor:Sprite Sprite]( [param:SpriteMaterial material]? )

Parameters

[parameter:SpriteMaterial material] - (optional) The material.

Creates eventDispatcher object. It needs to be call with '.call' to add the functionality to an object.

Properties

See the base [page:EventDispatcher EventDispatcher] class for common properties.

See the base [page:Object3D Object3D] class for common properties.

[property:boolean DefaultMatrixAutoUpdate]

[property:Vector3 DefaultUp]

[property:AnimationClip[] animations]

Array with animation clips.

[property:boolean castShadow]

Gets rendered into shadow map.

[property:Vector2 center]

[property:Object3D[] children]

Array with object's children.

[property:Material customDepthMaterial]

Custom depth material to be used when rendering to the depth map. Can only be used in context of meshes.
When shadow-casting with a DirectionalLight or SpotLight, if you are (a) modifying vertex positions in
the vertex shader, (b) using a displacement map, (c) using an alpha map with alphaTest, or (d) using a
transparent texture with alphaTest, you must specify a customDepthMaterial for proper shadows.

[property:Material customDistanceMaterial]

Same as customDepthMaterial, but used with PointLight.

[property:boolean frustumCulled]

When this is set, it checks every frame if the object is in the frustum of the camera before rendering the object.
If set to false the object gets rendered every frame even if it is not in the frustum of the camera.

[property:BufferGeometry geometry]

[property:number id]

Unique number of this object instance.

[property:any isObject3D]

Used to check whether this or derived classes are Object3Ds. Default is true.
You should not change this, as it is used internally for optimisation.

[property:any isSprite]

[property:Layers layers]

[property:SpriteMaterial material]

[property:Matrix4 matrix]

Local transform.

[property:boolean matrixAutoUpdate]

When this is set, it calculates the matrix of position, (rotation or quaternion) and scale every frame and also
recalculates the matrixWorld property.

[property:Matrix4 matrixWorld]

The global transform of the object. If the Object3d has no parent, then it's identical to the local transform.

[property:boolean matrixWorldNeedsUpdate]

When this is set, it calculates the matrixWorld in that frame and resets this property to false.

[property:Matrix4 modelViewMatrix]

[property:string name]

Optional name of the object (doesn't need to be unique).

[property:Matrix3 normalMatrix]

[property:Object3D parent]

Object's parent in the scene graph.

[property:Vector3 position]

Object's local position.

[property:Quaternion quaternion]

Object's local rotation as a Quaternion.

[property:boolean receiveShadow]

Material gets baked in shadow receiving.

[property:number renderOrder]

Overrides the default rendering order of scene graph objects, from lowest to highest renderOrder.
Opaque and transparent objects remain sorted independently though.
When this property is set for an instance of Group, all descendants objects will be sorted and rendered together.

[property:Euler rotation]

Object's local rotation (Euler angles), in radians.

[property:Vector3 scale]

Object's local scale.

[property:any type]

[property:Vector3 up]

Up direction.

[property:any userData]

An object that can be used to store custom data about the Object3d. It should not hold references to functions as these will not be cloned.

[property:string uuid]

[property:boolean visible]

Object gets rendered if true.
