GeometryComponent

The Geometry component.

Preview - ngx geometey




See the [page:GeometryComponent ngx3js docs] page for details.
See the [example:ngx_geometry ngx geometey] page for a live demo.
See the [example:ngx_font ngx font] page for a live font demo.
See the [example:ngx_curve ngx curve] page for a live curve demo.

A representation of mesh, line, or point geometry. Includes vertex positions, face indices, normals, colors, UVs, and custom attributes within buffers, reducing the cost of passing all this data to the GPU.
To read and edit data in BufferGeometry attributes, see [page:BufferAttribute BufferAttribute] documentation.
