[property:number angle]Input Public

Maximum angle of light dispersion from its direction whose upper bound is Math.PI/2(360).

[property:ThreeColor color]Input Public

Numeric value of the RGB component of the color. Default is 0xffffff.

[property:number decay]Input Public

The amount the light dims along the distance of the light. Default is 1.

[property:number distance]Input Public

Maximum range of the light. Default is 0 (no limit).

[property:ThreeColor groundColor]Input Public

hexadecimal color of the ground. Default is 0xffffff.

[property:number height]Input Public

height of the light. Default is 10.

[property:number intensity]Input Public

Numeric value of the light's strength/intensity. Default is 1.

[property:number penumbra]Input Public

Percent of the spotlight cone that is attenuated due to penumbra. Takes values between zero and 1. Default is zero.

[property:any renderTarget]Input Public

The renderTarget of light component

[property:any renderer]Input Public

The renderer of light component

[property:string sh]Input Public

An instance of [page:SphericalHarmonics3 SphericalHarmonics3].

[property:number shadowBias]Input Public

Shadow map bias, how much to add or subtract from the normalized depth when deciding whether a surface is in shadow.
The default is 0. Very tiny adjustments here (in the order of 0.0001) may help reduce artefacts in shadows

[property:number shadowCameraBottom]Input Public

Camera frustum bottom plane.

[property:number shadowCameraFar]Input Public

Camera frustum far plane. Default is *2000*.
Must be greater than the current value of [page:Camera.near Camera.near] plane.

[property:number shadowCameraFov]Input Public

Camera frustum vertical field of view, from bottom to top of view, in degrees. Default is *50*.

[property:number shadowCameraLeft]Input Public

Camera frustum left plane.

[property:number shadowCameraNear]Input Public

The shadowCameraNear of light component

[property:number shadowCameraRight]Input Public

Camera frustum right plane.

[property:number shadowCameraTop]Input Public

Camera frustum top plane.

[property:number shadowCameraZoom]Input Public

Gets or sets the zoom factor of the camera. Default is *1*.

[property:number shadowFocus]Input Public

Used to focus the shadow camera. The camera's field of view is set as a percentage of the spotlight's field-of-view. Range is [0, 1]. Default is 1.0.

[property:number shadowMapSize]Input Public

A [page:Vector2 Vector2] defining the width and height of the shadow map.

[property:number shadowMapSizeHeight]Input Public

A [page:Vector2 Vector2] defining the width and height of the shadow map.
vector2.height

[property:number shadowMapSizeWidth]Input Public

A [page:Vector2 Vector2] defining the width and height of the shadow map.
vector2.width

[property:number shadowRadius]Input Public

Setting this to values greater than 1 will blur the edges of the shadow.
High values will cause unwanted banding effects in the shadows - a greater [page:LightShadow.mapSize LightShadow.mapSize] will allow for a higher value to be used here before these effects become visible.
If [page:WebGLRenderer.shadowMap.type WebGLRenderer.shadowMap.type] is set to [page:Renderer PCFSoftShadowMap], radius has no effect and it is recommended to increase softness by decreasing [page:LightShadow.mapSize LightShadow.mapSize] instead.
Note that this has no effect if the [page:WebGLRenderer.shadowMap.type WebGLRenderer.shadowMap.type] is set to [page:Renderer BasicShadowMap].

[property:ThreeColor skyColor]Input Public

hexadecimal color of the sky. Default is 0xffffff.

The target of light component

The targetX of light component

The targetY of light component

[property:number targetZ]Input Public

The targetZ of light component

[property:AbstractTextureComponent texture]Input Public

The texture of light component

The type of light
