[property:number angle]

Maximum angle of light dispersion from its direction whose upper bound is Math.PI/2(360).

[property:ThreeColor color]

Numeric value of the RGB component of the color. Default is 0xffffff.

[property:number decay]

The amount the light dims along the distance of the light. Default is 1.

[property:number distance]

Maximum range of the light. Default is 0 (no limit).

[property:ThreeColor groundColor]

hexadecimal color of the ground. Default is 0xffffff.

[property:number height]

height of the light. Default is 10.

[property:number intensity]

Numeric value of the light's strength/intensity. Default is 1.

[property:number penumbra]

Percent of the spotlight cone that is attenuated due to penumbra. Takes values between zero and 1. Default is zero.

[property:any renderTarget]

Input of light component

[property:any renderer]

Input of light component

[property:string sh]

An instance of [page:SphericalHarmonics3].

[property:number shadowBias]

Shadow map bias, how much to add or subtract from the normalized depth when deciding whether a surface is in shadow.
The default is 0. Very tiny adjustments here (in the order of 0.0001) may help reduce artefacts in shadows

[property:number shadowCameraBottom]

Camera frustum bottom plane.

[property:number shadowCameraFar]

Camera frustum far plane. Default is *2000*.
Must be greater than the current value of [page:.near near] plane.

[property:number shadowCameraFov]

Camera frustum vertical field of view, from bottom to top of view, in degrees. Default is *50*.

[property:number shadowCameraLeft]

Camera frustum left plane.

[property:number shadowCameraNear]

Input of light component

[property:number shadowCameraRight]

Camera frustum right plane.

[property:number shadowCameraTop]

Camera frustum top plane.

[property:number shadowCameraZoom]

Gets or sets the zoom factor of the camera. Default is *1*.

[property:number shadowFocus]

Used to focus the shadow camera. The camera's field of view is set as a percentage of the spotlight's field-of-view. Range is [0, 1]. Default is 1.0.

[property:number shadowMapSize]

A [Page:Vector2] defining the width and height of the shadow map.

[property:number shadowMapSizeHeight]

A [Page:Vector2] defining the width and height of the shadow map.
vector2.height

[property:number shadowMapSizeWidth]

A [Page:Vector2] defining the width and height of the shadow map.
vector2.width

[property:number shadowRadius]

Setting this to values greater than 1 will blur the edges of the shadow.
High values will cause unwanted banding effects in the shadows - a greater [page:.mapSize mapSize]
will allow for a higher value to be used here before these effects become visible.
If [page:WebGLRenderer.shadowMap.type] is set to [page:Renderer PCFSoftShadowMap], radius has
no effect and it is recommended to increase softness by decreasing [page:.mapSize mapSize] instead.
Note that this has no effect if the [page:WebGLRenderer.shadowMap.type] is set to [page:Renderer BasicShadowMap].

[property:ThreeColor skyColor]

hexadecimal color of the sky. Default is 0xffffff.

[property:any target]

Input of light component

[property:number targetX]

Input of light component

[property:number targetY]

Input of light component

[property:number targetZ]

Input of light component

[property:AbstractTextureComponent texture]

Input of light component

[property:string type]

The type of light

Notice - case insensitive.

[page:PointLight PointLight] - THREE.PointLight - PointLight, Point
[page:RectAreaLight RectAreaLight] - THREE.RectAreaLight - RectAreaLight, RectArea
[page:SpotLight SpotLight] - THREE.SpotLight - SpotLight, Spot
[page:DirectionalLight DirectionalLight] - THREE.DirectionalLight - DirectionalLight, Directional
[page:HemisphereLight HemisphereLight] - THREE.HemisphereLight - HemisphereLight, Hemisphere
[page:LightProbe LightProbe] - THREE.LightProbe - LightProbe, Probe
[page:AmbientLight AmbientLight] - THREE.AmbientLight - AmbientLight, Ambient

[property:number width]

width of the light. Default is 10.
