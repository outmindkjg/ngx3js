color -- The desired color.

The first color used for grid elements. This can be a [page:Color Color], a hexadecimal value and an CSS-Color name. Default is 0x444444

The second color used for grid elements. This can be a [page:Color Color], a hexadecimal value and an CSS-Color name. Default is 0x888888

[property:boolean depthWrite]Input Public

Whether rendering this material has any effect on the depth buffer. Default is *true*.

X Direction from origin. Must be a unit vector.

Y Direction from origin. Must be a unit vector.

[property:number dirZ]Input Public

length of the arrow. Default is *1*.

[property:boolean materialTransparent]Input Public

Defines whether this material is transparent. This has an effect on rendering as transparent objects need special treatment and are rendered after non-transparent objects.
When set to true, the extent to which the material is transparent is controlled by setting it's .opacity property.
Default is false.

[property:Matrix4 matrix]Input Public

Update matrix for this helper

[property:number opacity]Input Public

Float in the range of *0.0* - *1.0* indicating how transparent the material is.
