OrthographicCamera

Camera that uses [link:https://en.wikipedia.org/wiki/Orthographic_projection orthographic projection].
In this projection mode, an object's size in the rendered image stays constant regardless of its distance from the camera.
This can be useful for rendering 2D scenes and UI elements, amongst other things.

See the [page:OrthographicCamera ngx3js docs] page for details.
See the [example:ngx_camera/OrthographicCamera ngx camera] page for a live demo.

The local transform matrix.

[property:boolean matrixAutoUpdate]Inherited

When this is set, it calculates the matrix of position, (rotation or quaternion) and scale every frame and also recalculates the matrixWorld property. Default is [page:Object3D.DefaultMatrixAutoUpdate Object3D.DefaultMatrixAutoUpdate] (true).

[property:Matrix4 matrixWorld]Inherited

The global transform of the object. If the Object3D has no parent, then it's identical to the local transform .matrix.

[property:Matrix4 matrixWorldInverse]Inherited

This is the inverse of matrixWorld. MatrixWorld contains the Matrix which has the world transform of the Camera.

[property:boolean matrixWorldNeedsUpdate]Inherited

When this is set, it calculates the matrixWorld in that frame and resets this property to false. Default is false.

[property:Matrix4 modelViewMatrix]Inherited

This is passed to the shader and used to calculate the position of the object.

[property:string name]Inherited

Optional name of the object (doesn't need to be unique). Default is an empty string.

[property:number near]

Camera frustum near plane. Default is 0.1.
The valid range is between 0 and the current value of the .far plane.
Note that, unlike for the [page:PerspectiveCamera PerspectiveCamera], 0 is a valid value for an OrthographicCamera's near plane.

[property:Matrix3 normalMatrix]Inherited

This is passed to the shader and used to calculate lighting for the object. It is the transpose of the inverse of the upper left 3x3 sub-matrix of this object's modelViewMatrix.
The reason for this special matrix is that simply using the modelViewMatrix could result in a non-unit length of normals (on scaling) or in a non-perpendicular direction (on non-uniform scaling).
On the other hand the translation part of the modelViewMatrix is not relevant for the calculation of normals. Thus a Matrix3 is sufficient.

[property:Object3D parent]Inherited

Object's parent in the [link:https://en.wikipedia.org/wiki/Scene_graph scene graph]. An object can have at most one parent.

[property:Vector3 position]Inherited

A [page:Vector3 Vector3] representing the object's local position. Default is (0, 0, 0).

[property:Matrix4 projectionMatrix]Inherited

This is the matrix which contains the projection.

[property:Matrix4 projectionMatrixInverse]Inherited

The inverse of projectionMatrix.

[property:Quaternion quaternion]Inherited

Object's local rotation as a [page:Quaternion Quaternion].

[property:boolean receiveShadow]Inherited

Whether the material receives shadows. Default is false.

[property:number renderOrder]Inherited

This value allows the default rendering order of [link:https://en.wikipedia.org/wiki/Scene_graph scene graph] objects to be overridden although opaque and transparent objects remain sorted independently. When this property is set for an instance of [page:Group Group], all descendants objects will be sorted and rendered together.
Sorting is from lowest to highest renderOrder. Default value is 0.

[property:number right]

Camera frustum right plane.

[property:Euler rotation]Inherited

Object's local rotation (see [link:https://en.wikipedia.org/wiki/Euler_angles Euler angles]), in radians.

[property:Vector3 scale]Inherited

The object's local scale. Default is [link: 1, 1, 1 page:Vector3 Vector3].

[property:number top]

Camera frustum top plane.

[property:any type]

[property:Vector3 up]Inherited

This is used by the .lookAt method, for example, to determine the orientation of the result.
Default is [page:Object3D.DefaultUp Object3D.DefaultUp] - that is, ( 0, 1, 0 ).

[property:any userData]Inherited

An object that can be used to store custom data about the Object3D. It should not hold references to functions as these will not be cloned.

[property:string uuid]Inherited

[link:http://en.wikipedia.org/wiki/Universally_unique_identifier UUID] of this object instance.
This gets automatically assigned, so this shouldn't be edited.

[property:any view]

Set by [page:OrthographicCamera.setViewOffset setViewOffset]. Default is null.

[property:boolean visible]Inherited

Object gets rendered if true. Default is true.

[property:number zoom]

Gets or sets the zoom factor of the camera. Default is 1.
