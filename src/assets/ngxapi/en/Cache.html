Cache

A simple caching system, used internally by [page:FileLoader FileLoader].

Preview - WebGL / geometry / text



Examples


[example:webgl_geometry_text WebGL / geometry / text]
[example:webgl_interactive_instances_gpu WebGL / interactive / instances / gpu]
[example:webgl_loader_ttf WebGL / loader / ttf]
To enable caching across all loaders that use [page:FileLoader FileLoader] THREE.Cache.enabled = true.

Properties

[property:boolean enabled]

Whether caching is enabled. Default is false.

[property:any files]

An [page:Object object] that holds cached files.

Methods

[method:void add]( [param:string key], [param:any file] )

Parameters

[parameter:string key] - The key to reference the cached file by.
[parameter:any file] - The file to be cached.

Adds a cache entry with a key to reference the file. If this key already holds a file, it is overwritten.

[method:void clear]( )

Remove all values from the cache.

[method:any get]( [param:string key] )

Parameters

[parameter:string key] - A string key

Get the value of key. If the key does not exist undefined is returned.

[method:void remove]( [param:string key] )

Parameters

[parameter:string key] - A string key that references a cached file.

Remove the cached file associated with the key.
