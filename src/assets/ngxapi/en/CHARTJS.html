[property:'center'|'start'|'end' align]? - The label alignment

[property:Scriptable color]? - Color of tick

[property:boolean display]? - If true, show tick labels.
[property:Scriptable font]? - see Fonts
[property:boolean includeBounds]? - Should the defined `min` and `max` values be presented as ticks even if they are not "nice".
[property:number labelOffset]? - Distance in pixels to offset the label from the centre point of the tick (in the x direction for the x axis, and the y direction for the y axis). Note?: this can cause labels at the edges to be cropped by the edge of the canvas
[property:any major]? - The major.
[property:number maxRotation]? - Maximum rotation for tick labels when rotating to condense labels. Note?: Rotation doesn't occur until necessary. Note?: Only applicable to horizontal scales.
[property:number minRotation]? - Minimum rotation for tick labels. Note?: Only applicable to horizontal scales.
[property:boolean mirror]? - Flips tick labels around axis, displaying the labels inside the chart instead of outside. Note?: Only applicable to vertical scales.
[property:number padding]? - Sets the offset of the tick labels from the axis
[property:number sampleSize]? - The number of ticks to examine when deciding how many labels will fit. Setting a smaller value will be faster, but may be less accurate when there is large variability in label length.
[property:Scriptable showLabelBackdrop]? - If true, draw a background behind the tick labels.
[property:Scriptable textStrokeColor]? - The color of the stroke around the text.
[property:Scriptable textStrokeWidth]? - Stroke width around the text.
[property:number z]? - z-index of tick layer. Useful when ticks are drawn on chart area. Values <= 0 are drawn under datasets, > 0 on top.
