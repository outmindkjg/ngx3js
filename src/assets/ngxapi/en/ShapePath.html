ShapePath

This class is used to convert a series of shapes to an array of [Path](https://outmindkjg.github.io/ngx3js-doc/#/docs/ngxapi/en/Path)s, for example an SVG shape to a path (see the example below).

Preview - geometry / extrude / shapes2

Constructor

[constructor:ShapePath ShapePath]( )

Creates a new ShapePath. Unlike a [page:Path Path], no points are passed in as the ShapePath is designed to be generated after creation.
