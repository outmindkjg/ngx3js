Fog color. Example: If set to black, far away objects will be rendered black.

[property:boolean debug]   = false Input Public

Debug this Object

[property:number density]   = 0.00025 Input Public

Defines how fast the fog will grow dense.
Default is 0.00025.

[property:boolean enabled]   = true Input Public

Enabled or Not

[property:number far]   = 1000 Input Public

The maximum distance at which fog stops being calculated and applied. Objects that are more than 'far' units away from the active camera won't be affected by fog.
Default is 1000.

[property:string id]   = '' Protected

The Id of abstract subscribe component

[property:number near]   = 1 Input Public

The minimum distance to start applying fog. Objects that are less than 'near' units from the active camera won't be affected by fog.
Default is 1.

[property:EventEmitter onDestory]   = ... Output Public
