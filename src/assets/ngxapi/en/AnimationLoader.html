AnimationLoader

Class for loading [page:AnimationClip AnimationClips] in JSON format.
This uses the [page:FileLoader FileLoader] internally for loading files.

Code Example



// instantiate a loader
const loader = new THREE.AnimationLoader();
// load a resource
loader.load(
'animations/animation.js', // resource URL
function ( animations ) { // onLoad callback
// animations is an array of AnimationClips
},
function ( xhr ) { // onProgress callback
console.log( (xhr.loaded / xhr.total * 100) + '% loaded' );
},
function ( err ) { // onError callback
console.log( 'An error happened' );
}
);
