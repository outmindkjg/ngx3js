Layers

A [page:Layers Layers] object assigns an [page:Object3D Object3D] to 1 or more of 32 layers numbered 0 to 31 - internally the layers are stored as a [link:https://en.wikipedia.org/wiki/Mask_(computing bit mask]), and by default all Object3Ds are a member of layer 0.
This can be used to control visibility - an object must share a layer with a [page:Camera camera] to be visible when that camera's view is rendered.
All classes that inherit from [page:Object3D Object3D] have an [page:Object3D.layers Object3D.layers] property which is an instance of this class.
