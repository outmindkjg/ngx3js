Constructor

[constructor:NgxStarGeometry NgxStarGeometry]( [param:number radius]?, [param:number segments]?, [param:number thetaStart]?, [param:number thetaLength]? )Overwrite

Parameters

[parameter:number radius] - (optional) Radius of the circle, default = 1.
[parameter:number segments] - (optional) Number of segments (triangles), minimum = 3, default = 8.
[parameter:number thetaStart] - (optional) Start angle for first segment, default = 0 (three o'clock position).
[parameter:number thetaLength] - (optional) The central angle, often called theta, of the circular sector. The default is 2*Pi, which makes for a complete circle.

Creates eventDispatcher object. It needs to be call with '.call' to add the functionality to an object.
