Fog

This class contains the parameters that define linear fog, i.e., that grows linearly denser with the distance.

Constructor

[constructor:Fog Fog]( [param:ColorRepresentation color], [param:number near]?, [param:number far]? )

Parameters

[parameter:ColorRepresentation color] - The color.
[parameter:number near] - (optional) The near.
[parameter:number far] - (optional) The far.

Properties

See the base [page:FogBase FogBase] class for common properties.

[property:Color color]

Fog color. Example: If set to black, far away objects will be rendered black.

[property:number far]

The maximum distance at which fog stops being calculated and applied. Objects that are more than 'far' units away from the active camera won't be affected by fog.
Default is 1000.

[property:any isFog]

[property:string name]

Optional name of the object (doesn't need to be unique). Default is an empty string.

[property:number near]

The minimum distance to start applying fog. Objects that are less than 'near' units from the active camera won't be affected by fog.
Default is 1.
