Class ListenerComponent

The Listener component.

See the ngx3js docs page for details.

The [name] represents a virtual listener of the all positional and non-positional audio effects in the scene. A three.js application usually creates a single instance of [name]. It is a mandatory construtor parameter for audios entities like Audio and PositionalAudio. In most cases, the listener object is a child of the camera. So the 3D transformation of the camera represents the 3D transformation of the listener.

<ngx3js-camera>
    <ngx3js-listener></ngx3js-listener>
</ngx3js-camera>
